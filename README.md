# data-analysis

Starter template for small Python data analysis projects using Pandas and Jupyter.

## Project structure

- `analysis.ipynb` - main notebook for loading data and exploring it
- `functions.py` - reusable helper functions, including `read_data()`
- `data/` - place datasets here
- `requirements.txt` - Python dependencies for the project

## Getting started

1. Create and activate a virtual environment.
2. Install the dependencies:

```bash
pip install -r requirements.txt
```

3. Add your data file to the `data/` folder.
4. Open `analysis.ipynb` in Jupyter and update the filename if needed.

## Included example

The template includes `data/sample.csv` so the notebook runs immediately.
Replace it with your own dataset when you start a real analysis.
